# Amazon Mini Automation Project

A simple UI automation project for Amazon.com using **Java**, **Selenium WebDriver**, **Cucumber (BDD)**, **TestNG**, and **Maven**. The framework covers two core flows: **Login** and **Product Search**, following a clean, maintainable structure suitable for portfolio and interview use.

---

## Tools & Technologies
Java • Selenium WebDriver • Cucumber (Gherkin) • TestNG • Maven

---

## Project Structure
src/test/java  
• features — Gherkin scenarios  
• steps — Step definitions   
• runner — TestNG + Cucumber runner  
pom.xml — Dependencies  
testng.xml — Test suite

---

## Automated Scenarios
**Login**  
 **Product Search**  
  
---

## Running Tests
Maven command:  
`mvn clean test`  

Run specific suite:  
`mvn test -DsuiteXmlFile=testng.xml`

---

## Purpose
This project demonstrates practical automation skills using BDD, Selenium, and TestNG with a simple, scalable structure for real e‑commerce flows.
